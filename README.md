#### lowercase ui plugin for betterdiscord
###### description:
a plugin for lowercase discord ui with customization options. toggle off message content, user display, and server related info.
###### github:
- [betterdiscord](https://github.com/BetterDiscord) - required
- [lowercase ui plugin](https://github.com/thornling/lowercase-ui-plugin) - plugin files
- [lowercase ui theme](https://github.com/thornling/lowercase-ui-for-betterdiscord) - a simpler alternative
###### installation:
1. this plugin works with betterdiscord
2. settings > betterdiscord > plugins
3. folder icon to open local plugins folder
4. move or copy bd-lowercase-ui.plugin.js there
5. refresh plugins page if necessary and toggle on
